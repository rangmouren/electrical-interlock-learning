# ⚡ 电气控制连锁技术 — 交互式学习网站

从零开始，系统学习**自锁（Self-Lock）**、**互锁（Interlock）** 和 **复合连锁（Composite Interlock）** 电路。

## 📸 预览

直接在浏览器中打开 `index.html` 即可使用，无需任何服务器或依赖。

## 📚 学习路线

| # | 章节 | 内容 |
|---|------|------|
| 1 | 📘 绪论：什么是连锁控制 | 连锁概念、三种连锁形式对比、应用场景 |
| 2 | 🔧 基础元件与符号 | 按钮、接触器、继电器、NO/NC 区分 |
| 3 | ⚡ 基本电路原理 | 启动/停止电路、梯形图规则、与或非记忆逻辑 |
| 4 | 🔒 自锁电路理论 | 自锁工作原理、三步曲、停止按钮设计 |
| 5 | 🧪 **自锁交互实验** | 点击按钮仿真自锁保持过程 |
| 6 | 📝 自锁小测验 | 4 道选择题 + 解析 |
| 7 | 🚫 互锁电路理论 | 电机正反转控制、电气/机械/双重互锁 |
| 8 | 🧪 **互锁交互实验** | 模拟互锁封锁效果 |
| 9 | 📝 互锁小测验 | 4 道选择题 + 解析 |
| 10 | 🔄 复合连锁电路 | 多层保护、时间/行程/安全连锁 |
| 11 | 🧪 **复合连锁实验** | 电路分析挑战 + 故障场景推演 |
| 12 | 📝 复合连锁小测验 | 4 道选择题 + 解析 |
| 13 | 🏭 实际工程应用 | 传送带系统、电梯、星三角启动案例 |

## ✨ 特性

- **SVG 梯形图可视化** — 每个电路都配有清晰的梯形图
- **交互式电路仿真** — 自锁保持实验、互锁封锁实验，点击操作实时反馈
- **章节测验 + 综合测验** — 每题均有详细解析
- **学习进度追踪** — 浏览器 localStorage 自动保存进度
- **深色模式** — 右上角 🌓 一键切换
- **移动端适配** — 响应式布局，手机侧边栏菜单

## 🚀 使用方式

直接打开 `index.html` 即可：

```bash
# 克隆仓库
git clone https://github.com/rangmouren/electrical-interlock-learning.git

# 打开
open electrical-interlock-learning/index.html
```

或直接在浏览器中点击 `index.html` 文件。

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript（零依赖）
- SVG 绘制电路梯形图
- localStorage 持久化进度和主题
- 响应式设计支持移动端
