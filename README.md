# ⚡ 电气控制连锁技术 — 交互式学习网站

从零开始，系统学习**自锁（Self-Lock）**、**互锁（Interlock）** 和 **复合连锁（Composite Interlock）** 电路。

**v2.0 已发布** — 代码模块化重构，新增交互实验、键盘快捷键、故障场景推演等

## ✨ 特性

- **SVG 梯形图可视化** — 每个电路配有清晰的梯形图，带颜色标识电源/信号路径
- **交互式电路仿真** — 自锁保持实验、互锁封锁实验，点击操作实时反馈
- **章节测验 + 综合测验** — 每题均有详细解析
- **学习进度追踪** — localStorage 持久化保存进度
- **深色模式** — 右上角一键切换（Alt+D）
- **键盘快捷键** — ← → 翻页，Alt+D 切换主题，H 显示帮助
- **故障场景推演** — 复合连锁章节提供3种故障场景分析
- **移动端适配** — 响应式布局，手机侧边栏菜单

## 🚀 使用方式

直接打开 `index.html` 即可：

```bash
# 克隆仓库
git clone https://github.com/rangmouren/electrical-interlock-learning.git

# 打开
open electrical-interlock-learning/index.html
```

或直接在浏览器中点击 `index.html` 文件。

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript（零依赖）
- SVG 绘制电路梯形图
- localStorage 持久化进度和主题
- 响应式设计支持移动端
